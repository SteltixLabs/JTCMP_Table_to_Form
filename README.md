# form-editable-table

Oracle JET component. Consists of a filterable table, where any row can be selected so its cells can be 
individually edited.

## Example

In `example/js` are some files from the `js` folder of an Oracle JET QuickStart Basic sample site, 
with the form-editable-table component embedded into the "dashboard" section of the site. 
Component code is located in `example/js/composites/form-editable-table`. The code for embedding 
the component into the site is in `example/js/views/dashboard.html` and 
`example/js/viewModels/dashboard.js`
